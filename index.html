A simple web Celsius and Fahrenheit temperature conversion.
This website was developed during practical coding exercises as a mini project of the Basic Software Engineering Course at revoU.

*enter a value in either Celsius or Fahrenheit field.

Calculation Formula:

Celsius to Fahrenheit: °C = (9/5 x °C) + 32

Fahrenheit to Celsius: °F = 5/9 x (°F - 32)
